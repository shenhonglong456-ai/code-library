# B01 XHS 3 图套 + 视频号 30 秒短视频 · 生成预备

## 一、XHS 3 张图（baoyu-xhs-images）

### 推荐配置

| 参数 | 推荐值 | 备选 |
|------|-------|------|
| style | `warm`（方法论类内容，温暖靠谱）| `bold` / `notion` |
| layout | `dense`（3 张要装下方法论，密度高一些）| `list` |
| 配色 | 米黄 + 深蓝（方法论靠谱感）| 红黑白 |

### 3 张图文案

#### 图 1（封面 - 提问式钩子）

**顶部大字**：
> 快递涨价落地，
> 你真金白银多收到多少？

**副标**：
> 如果答案是"很少"或"没多少"，
> 那今年你最该盯这 3 件事。

**视觉**：老板皱眉看账本；问号。

---

#### 图 2（核心方法论）

**顶部大字**：
> 不是等涨价，是**主动找利润**

**正文（3 个抓手）**：

✅ **盯账本**：不看单量，看单票净收益
看每单到手多少，不是派了多少。

✅ **盯退货**：退货件不只是成本，是生意
每天几千单退货，账算明白，就是利润。

✅ **盯客户结构**：20% 客户赚你 80% 利润
该留的客户留住，该换的区域换掉。

**视觉**：三栏列式；✅ 绿色勾醒目。

---

#### 图 3（CTA 钩子）

**顶部大字**：
> 你不主动算，
> 钱就不会主动来。

**中部**：
> 点赞收藏，
> 评论区留"**账本**"，
>
> 免费发你一份 **《快递网点单票净收益 / 退货件利润 / 客户结构 三表套装》**

**底部**：
> 或扫码加微（客服二维码占位）

**视觉**：三本账本图标 + 二维码。

## 二、视频号 30 秒短视频（video-producer）

### 推荐配置

| 参数 | 推荐值 | 备选 |
|------|-------|------|
| 时长 | 30 秒（±3 秒）| 60 秒加深度 |
| 画幅 | 9:16 竖屏 | 16:9 横屏（不建议，视频号偏竖）|
| TTS 声线 | **CosyVoice longshu**（已在 ~/.baoyu-skills/.env）| MiniMax 备选 |
| 视觉模式 | **V1 期推荐：AI 图+动态字幕+口播**（不用真人，不用数字人，最省事）| 真人出镜 / 数字人分身（V1 期不启用）|
| 背景音 | 轻快 BGM（无版权）| |
| 字幕 | 全程打在屏幕下 1/3，字号大 | |

### 脚本分镜（video-producer 可直接用）

| 时间 | 画面 | 口播 | 字幕 |
|------|------|------|------|
| 0-3 秒 | 老板皱眉看手机（AI 图）"XX 快递涨价"新闻 | "快递涨价落地了，你兜里多了吗？" | 同口播 |
| 3-10 秒 | 账本特写 + 算账的手（AI 图） | "总部涨价是真的，但钱没走到网点。对客户涨了 0.3 毛，返给你多少？很多省份根本没变。" | 同口播 |
| 10-20 秒 | 三屏分屏："单票净收益" "退货件利润" "客户结构" | "今年作为网点老板，别把希望只压在涨价上。真正该盯的是 3 件事：第一，单票净收益——不是派了多少单，是每单到手多少。第二，退货件的账——退货件本身就是生意。第三，客户结构——20% 的客户赚你 80% 的利润。" | 3 个关键词放大 |
| 20-30 秒 | 二维码特写 + 大字"评论区留'账本'" | "评论区留'账本'两个字，我免费发你一份《单票净收益 / 退货件 / 客户结构 三表套装》。涨价不主动来，钱也不主动来。" | 同口播 + CTA 箭头 |
